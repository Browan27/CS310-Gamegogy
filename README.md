Gamegogy
=====

CS310 Scrum Project

### Gradle

run terminal app
```bash
 // run with gradle
$ gradle run
 // build and run the JAR using java
$ gradle build && java -jar ./build/libs/CS310-Gamegogy-0.0.1.jar
```

create coverage reports
```bash
$ gradle jacocoTestReport
```

run junit tests
```bash
$ gradle test
```

run robot acceptance tests
```bash
$ gradle runrobot
```
